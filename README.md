判別分析 GUI ツール 配布説明

1. 概要
このツールは、学習用ファイルとテスト用ファイルを読み込み、
判別分析 (LDA) によりテストデータへ予測ラベルを付与する Windows 用 GUI アプリです。

2. 対応ファイル
- 入力対応形式: csv, xlsx
- 出力形式:
  - レポート: csv
  - 予測結果: テスト入力が csv の場合は csv、xlsx の場合は xlsx

3. 起動方法
- DiscriminantAnalysisGUI.exe をダブルクリックしてください。

4. 使い方
- 学習ファイルを選択
- テストファイルを選択
- 必要に応じて出力フォルダを選択
- 実行 ボタンを押す

5. 入力データの前提
- 学習ファイル:
  - 1列目: ID
  - 2列目以降: 特徴量
  - 最終列: ラベル
- テストファイル:
  - 1列目: ID
  - 2列目以降: 特徴量
  - 最終列が label / target / cluster の場合はラベル列として扱います

6. 出力ファイル
- 既存ファイルを上書きしないよう、出力ファイル名の末尾に月日時分を付けます
- 同じ分に複数回実行した場合は _01 のような連番が付きます

7. 出力される主なファイル
- lda_report_月日時分.csv
- test_with_prediction_月日時分.csv または xlsx

8. 注意事項
- 特徴量列には数値を入力してください
- 学習データとテストデータの特徴量数は一致している必要があります
- 学習データのラベル列には 2 つ以上のグループが必要です

9. 配布物
- 本アプリは Windows 用 exe です
- インストール不要で実行できます

10. ビルド情報
- GUI 版
- onefile 形式
- 対応形式: csv, xlsx
