discriminant_analysis.py README

部内共有用の判別分析ツール通常版です。学習用データとテスト用データを読み込み、線形判別分析（LDA）でテスト用データに予測ラベルを付与します。

対象ファイル

  • スクリプト: discriminant_analysis.py
  • 実行ファイル: DiscriminantAnalysis.exe
  • GUI起動用バッチ: run_gui.bat
  • ビルド用バッチ: build_exe.bat

できること

  • CSV（.csv）とExcel（.xlsx）を入力できます。
  • GUIとコマンドラインの両方で実行できます。
  • 学習データの判別率、クロス集計、係数、切片、標準化情報をレポートに出力します。
  • テストデータに predicted_label 列を追加したファイルを出力します。
  • 出力ファイル名には日時が付き、既存ファイルを上書きしません。

入力データ形式

学習用データ

列 内容
1列目 ID
2列目以降 特徴量（数値）
最終列 ラベルまたはクラスター

学習用データのラベルは2種類以上必要です。

テスト用データ

列 内容
1列目 ID
2列目以降 特徴量（数値）
最終列 任意

テスト用データの最終列名が cluster、label、target のいずれかの場合、その列は正解ラベル列として扱われ、予測用の特徴量から除外されます。それ以外の場合は、2列目以降すべてを特徴量として扱います。

入力時の注意

  • 学習用データとテスト用データの特徴量数は一致している必要があります。
  • 特徴量列はすべて数値にしてください。
  • 空行や説明行が先頭にある場合でも、最初に見つかったヘッダー行から読み込みます。
  • CSVはUTF-8 BOM付きにも対応しています。
  • Excelは先頭シートを読み込みます。

GUIで実行する

実行ファイルを使う場合:

.\DiscriminantAnalysis.exe

Pythonスクリプトを使う場合:

python .\discriminant_analysis.py

または:

.\run_gui.bat

画面で学習用ファイル、テスト用ファイル、必要に応じて出力フォルダを選択し、実行ボタンを押します。

コマンドラインで実行する

Excelの例:

python .\discriminant_analysis.py --train .\data.xlsx --test .\test.xlsx --output-dir .\output

CSVの例:

python .\discriminant_analysis.py --train .\data.csv --test .\test.csv --output-dir .\output

主な引数:

引数 内容
--train 学習用ファイルのパス
--test テスト用ファイルのパス
--output-dir 出力先フォルダ
--output-report レポート出力ファイル名
--output-test 予測結果出力ファイル名
--gui GUIを起動

--output-report と --output-test を指定した場合も、指定したファイル名に日時が付与されます。

出力ファイル

出力先フォルダを指定しない場合、テスト用データと同じフォルダに保存されます。ファイル名には MMDDHHMM 形式の日時が付きます。同名ファイルがある場合は _01、_02 のような連番が付きます。

出力 内容
lda_report_日時.csv または lda_report_日時.xlsx 学習判別率、クロス集計、クラス事前確率、係数、切片、標準化平均、標準化分散、テストデータの予測グループ分布
<テストファイル名>_with_prediction_日時.csv または .xlsx テスト用データに predicted_label 列を追加した結果

出力形式は、入力したテスト用ファイルの拡張子に合わせてCSVまたはExcelになります。

レポートの見方

  • 学習判別率: 学習データに対する予測一致率です。
  • 学習データクロス集計: 実際のラベルと予測グループの対応表です。
  • クラス事前確率: 学習データ内の各クラスの比率です。
  • coef_* と intercept: 判別式で使用した係数と切片です。
  • 標準化平均 と 標準化分散: 特徴量の標準化に使用した値です。
  • テストデータ予測グループ集計分布: 予測されたグループごとの件数と比率です。

ビルドする場合

.\build_exe.bat

ビルドには pyinstaller、numpy、openpyxl を使用します。作成された実行ファイルは dist\DiscriminantAnalysis.exe に出力されます。

よくあるエラーと確認点

エラー内容 確認点
ファイルが見つからない 指定したファイルパスが正しいか確認してください。
未対応形式 通常版は .csv と .xlsx に対応しています。
特徴量列には数値を入力してください 特徴量列に文字列や空欄が混ざっていないか確認してください。
ラベル列には2つ以上のグループが必要 学習用データの最終列に2種類以上のラベルがあるか確認してください。
特徴量数が一致しない 学習用データとテスト用データで特徴量列の数をそろえてください。
